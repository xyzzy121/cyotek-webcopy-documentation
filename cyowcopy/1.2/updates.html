Checking for updates

Manually checking for updates

  • Open the Help menu and choose Check for updates
  • The Cyotek website will be queried for the latest version of WebCopy
  • If an update is available, a prompt will appear to navigate to the Cyotek website to download the new version

Automatically checking for updates

If the automatic update check option is set, WebCopy will automatically check for updates once a day. If an update is found, a notification dialog will be displayed containing a download link.

When checking for updates the application code and version is transferred, no other information is sent.
