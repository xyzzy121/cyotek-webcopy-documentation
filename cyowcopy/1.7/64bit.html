32 vs 64-bit

WebCopy includes both 32bit and 64bit versions of its primary components. The 64bit versions allow for greater memory usage, but also consume more memory for all actions. Cyotek recommends using the 32bit versions unless specific issues are encountered; please contact support if you are running into issues with the 32bit clients.

32-bit Filename64-bit FilenameDescription
cyowcopy.execyowcopy-x64.exeWebCopy GUI client
wcopy.exewcopy-x64.exeWebCopy CLI client
lnkchkgui.exelnkchkgui-x64.exeLink Checker GUI client
lnkchk.exelnkchk-x64.exeLink Checker CLI client
xpathtst.exeN/AXPath Testing tool
uritest.exeN/AURI Testing tool

WebCopy setup does not install the 64 bit versions of tools by default.
