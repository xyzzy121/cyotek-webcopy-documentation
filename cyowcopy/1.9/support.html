Getting support

If you are having difficulty using WebCopy or have questions on how to use a particular feature, limited support is available to all users of the latest version of this product.

Using the latest version

If you are experiencing a problem using the software, please ensure you are using the latest version of WebCopy, Cyotek frequently issues updates and may have already addressed your issue. In addition, we are unable to offer support for older versions of this software.

Accessing Support Resources from an Internet Browser

  1. Launch your Internet Browser
  2. Enter this URL in the address bar: https://www.cyotek.com/support

Accessing Support Resources from within the application

  1. From the Help menu, click Technical Support
  2. The support website will be opened in your default Internet Browser

To submit a Support Request

Important

This feature will only appear if the Submit Feedback extension is installed and enabled.

  1. From the Help menu, click Feedback, then Send a Support Request
  2. Enter the question or issue you are having. Please enter as many details as possible
  3. To include a screenshot, check the Include a screenshot option
  4. If you'd like a response from the Cyotek Team, fill in your email address
  5. Click Submit to submit your request

Important

Screenshots capture all visible windows on your desktop. You should ensure that no private or confidential information is visible before using the feedback utility.
