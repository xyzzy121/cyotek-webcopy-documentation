Crawling additional root URL's

If there are sections of your site which do not have any inbound links, but which you still wish crawled, you can specify additional seed URL's for the crawl process.

To customize additional root URL's

  • From the Project Properties dialog, click the Additional Root URL's category

Adding a new URL

  1. In the URI field, enter the URl
  2. Click the Add button

Deleting a URL

  1. Select the URL you wish to remove from the list
  2. Click the Delete button
